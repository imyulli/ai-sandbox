# Task2.2.1 Javascript New Function Generation Template

*The task description covers only the initial requirements. For training purposes, the task is similar to real-world situations, with the developer learning about specific requirements during the development process. Use the test results to examine and identify all requirements.*

This task requires you to engage with LLM to get JavaScript function implementations for a set of requirements:

* Calculate the sum of all prime numbers up to a given number n:
function sumOfPrimes(n: number): number
* Convert a string to title case (first letter of each word capitalized):
function toTitleCase(input: string): string
* Given an array of strings ["eat","tea","tan","ate","nat","bat"], group the anagrams together. The result can be returned in any order, as an example: [["bat"],["nat","tan"],["ate","eat","tea"]]

It is recommended that you complete this task with the help of [EPAM Dial](https://chat.lab.epam.com/).
